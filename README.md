# Construct-Koszul-Filtrations
Three algorithms that aid in the construction of Koszul filtrations. 
in standard graded k-algebras.

Given a set of linear forms L, we: 

(1) Generate candidate ideals for a Koszul filtration. 

(2) Confirm or deny the existence of a Koszul filtration with respect to the given list L.

(3) Construct linear flags for a given ideal.


For more info, check out Constructing Koszul filtrations: existence and non-existence for G-quadratic algebras by Berghofer, Nicklasson, Thompson and Westerbäck, arXiv: 2602.06490.
